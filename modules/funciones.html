Funciones: Tu amigo el ayudante

Imagina que tienes un amigo muy inteligente llamado Calculadora. Si de la nada le gritas: "¡Suma!", tu amigo te va a mirar confundido y te dirá: "¿Sumar qué? Necesito los números".

En programación, una función es como ese amigo al que le encargas una tarea. Para que pueda hacerla, necesitas darle contexto. Ese contexto se lo entregas a través de los paréntesis () y se llaman parámetros.

Sintaxis: Usamos la palabra def (de definir) para crear a nuestro "amigo", le ponemos un nombre, y entre paréntesis le decimos qué información necesita para trabajar. 
# 1. Creamos a nuestro amigo "sumar_amigo" y le decimos que necesita dos números
def sumar_amigo(numero1, numero2):
    resultado = numero1 + numero2
    # El 'return' es cuando tu amigo te entrega un papelito con el resultado final
    return resultado

# 2. Le pedimos a nuestro amigo que trabaje dándole el contexto (5 y 3)
mi_suma = sumar_amigo(5, 3)

# 3. Ahora podemos hacer lo que queramos con ese resultado
print(f"Mi amigo me devolvió el número: {mi_suma}")

La diferencia entre print() y return

Este es el error más común: pensar que imprimir es lo mismo que devolver.

  • print() es como si tu amigo gritara el resultado para que todos lo escuchen. Tú no puedes guardar un grito en tu bolsillo.
  • return es como si tu amigo escribiera el resultado en un papel y te lo entregara en la mano. Ese papel sí lo puedes guardar en una variable para usarlo más tarde.
Práctica 1: El Cajero Automático

Crea un script en Python que haga lo siguiente:

  1. Crea una función llamada aplicar_descuento que reciba dos parámetros: precio_original y descuento.
  2. La función debe calcular cuánto dinero se descuenta, restárselo al precio original, y hacer un return del precio final.
  3. Fuera de la función, crea una variable llamada precio_televisor con el valor 1000.
  4. Usa tu función para aplicarle un descuento de 200, guarda el resultado en una variable e imprímelo.
solucion_funciones.py 
# 1 y 2. Definimos la función y le pasamos sus parámetros (contexto)
def aplicar_descuento(precio_original, descuento):
    precio_final = precio_original - descuento
    return precio_final  # Entregamos el papelito con el resultado

# 3. Definimos nuestro precio
precio_televisor = 1000

# 4. Llamamos a nuestro "amigo" para que haga el cálculo
precio_a_pagar = aplicar_descuento(precio_televisor, 200)

print(f"El televisor costaba ${precio_televisor}.")
print(f"Con el descuento, debes pagar: ${precio_a_pagar}")

Recursividad: Una función que se llama a sí misma

La recursividad suena a magia negra, pero es simple: es cuando una función se utiliza a sí misma dentro de su propio código para resolver un problema más pequeño.

Para que esto no se repita para siempre (como un espejo frente a otro espejo), toda función recursiva debe tener un caso base (una condición donde se detiene).

# Ejemplo de recursividad: Cuenta regresiva
def cuenta_regresiva(numero):
    if numero == 0:  # ¡Este es el CASO BASE! Aquí se detiene la recursión.
        print("¡Despegue!")
    else:
        print(numero)
        # La función se llama a sí misma, pero pasándole un número más pequeño
        cuenta_regresiva(numero - 1)

# Llamamos a la función
cuenta_regresiva(5)
Práctica 2: El Clásico Fibonacci

La secuencia de Fibonacci es una serie de números donde el siguiente número es la suma de los dos anteriores: 0, 1, 1, 2, 3, 5, 8, 13...

Tu objetivo es crear una función recursiva llamada fibonacci(n) que te devuelva el número que está en la posición n de la secuencia.

Pistas de los Casos Base:

  • Si n es 0, debe devolver 0.
  • Si n es 1, debe devolver 1.
  • Si es mayor, debe devolver la suma de sí misma llamando a (n - 1) y (n - 2).
solucion_fibonacci.py 
def fibonacci(n):
    # Casos base (para que la recursión se detenga)
    if n == 0:
        return 0
    elif n == 1:
        return 1
    # Caso recursivo: sumamos los dos números anteriores de la secuencia
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

# Probamos la función pidiendo el número en la posición 6 (debe dar 8)
resultado = fibonacci(6)
print(f"El número en la posición 6 de la secuencia es: {resultado}")
