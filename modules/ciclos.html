Ciclos: While y For

Aprende a repetir tareas automáticamente y a controlar el flujo de tu programa.

Imagina que tienes que imprimir los números del 1 al 100. Podrías escribir print(1), print(2)... ¡cien veces! O podrías usar un ciclo (también llamado bucle o loop) para decirle a Python: "Haz esto 100 veces por mí".

El ciclo while

Usamos un ciclo while (que en español significa "mientras") cuando queremos que el código se repita mientras una condición sea verdadera. No sabemos exactamente cuántas veces va a ejecutarse, ya que depende de lo que pase en el programa.

¡Cuidado con los bucles infinitos! Si la condición del while nunca se vuelve falsa, tu programa se quedará atascado repitiendo para siempre. 
# Una cuenta regresiva con while
contador = 5

print("Comenzando la cuenta regresiva:")

while contador > 0:
    print(contador)
    contador = contador - 1  # ¡Crucial! Le restamos 1 para que en algún momento llegue a 0

print("¡Despegue!")

Detener ciclos: break

A veces queremos detener un ciclo antes de que su condición se vuelva falsa, o queremos crear un ciclo infinito a propósito (while True) y detenerlo solo cuando pase algo específico. Para eso usamos la instrucción break.

# Ciclo infinito controlado por un break
while True:
    respuesta = input("Escribe 'salir' para detener el programa: ")
    
    if respuesta == "salir":
        print("Deteniendo el programa...")
        break  # Esto rompe el ciclo y salta a la línea de abajo
    else:
        print(f"Escribiste {respuesta}. Seguimos atrapados en el ciclo.")

print("¡Saliste del ciclo con éxito!")

El ciclo for

Una vez que entiendes el while, el for te resultará muy cómodo. Lo usamos cuando sabemos exactamente cuántas veces queremos repetir algo, o cuando queremos recorrer los elementos de una colección (como una lista).

El truco de range(): En Python, la función range(3) genera una secuencia de 3 números (0, 1 y 2). Es el mejor compañero del ciclo for. 
# Repetir algo 3 veces usando range()
print("Comenzando el ciclo for:")

for numero in range(3):
    print(f"Esta es la repetición número {numero}")
    
print("¡Ciclo terminado!")

El Desafío: "El guardia de seguridad"

Escribe un programa usando un ciclo while True y la instrucción break que actúe como una contraseña.

  • El programa debe pedirle al usuario repetidamente que ingrese la palabra secreta.
  • Si el usuario escribe "python123", el programa le da la bienvenida y rompe el ciclo.
  • Si escribe otra cosa, le avisa que es incorrecto y le vuelve a preguntar.
solucion.py 
# Iniciamos un ciclo infinito
while True:
    palabra_secreta = input("Ingresa la contraseña para entrar: ")
    
    if palabra_secreta == "python123":
        print("¡Acceso concedido! Bienvenido al sistema.")
        break  # Rompemos el ciclo porque adivinó la clave
    else:
        print("Contraseña incorrecta, intenta de nuevo.\n")
