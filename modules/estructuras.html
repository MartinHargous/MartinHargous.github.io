Estructuras de Datos

Aprende a organizar y agrupar grandes cantidades de información usando Listas y Diccionarios.

Hasta ahora, cada variable guardaba un solo dato (como edad = 25). Pero, ¿qué pasa si queremos guardar las edades de 100 estudiantes? ¿O toda la información de un usuario (nombre, correo, edad) junta? Para eso existen las estructuras de datos.

Listas (Arrays)

Una lista es una colección ordenada de elementos. Imagínala como un estante con casilleros numerados. Puedes guardar lo que quieras dentro, separados por comas y encerrados entre corchetes [].

Regla de oro de la programación: ¡Se empieza a contar desde el cero! El primer elemento de una lista está en la posición (índice) 0, el segundo en el 1, y así sucesivamente. 
# Crear una lista de frutas
frutas = ["Manzana", "Platano", "Naranja"]

# Imprimir toda la lista de golpe
print(frutas)

# Acceder a un elemento específico (el primero)
print(f"Mi fruta favorita es la {frutas[0]}")

# Agregar un nuevo elemento al final de la lista con .append()
frutas.append("Uva")
print("Lista actualizada:", frutas)
El poder del ciclo For con las Listas

¿Recuerdas el ciclo for que vimos antes? Su uso más común y poderoso en Python es recorrer listas. En lugar de imprimir cada elemento uno por uno usando índices (frutas[0], frutas[1]...), podemos hacer que Python lo haga automáticamente.

frutas = ["Manzana", "Platano", "Naranja", "Uva"]

# Le decimos a Python: "Por cada 'fruta' individual dentro de la lista 'frutas', haz esto:"
for fruta in frutas:
    print(f"Me gusta comer {fruta}")
    
# Salida:
# Me gusta comer Manzana
# Me gusta comer Platano
# ... y así con todas.
Práctica 1: Lista de Tareas

Crea un script en Python que haga lo siguiente:

  1. Crea una lista llamada tareas con tres cosas que debas hacer hoy.
  2. Usa .append() para agregar una cuarta tarea a la lista.
  3. Imprime un mensaje que diga cuál es la primera tarea de tu lista (usando su índice).
  4. Utiliza un ciclo for para imprimir todas tus tareas una por una en forma de lista vertical.
solucion_listas.py 
# 1. Creamos la lista
tareas = ["Estudiar Python", "Hacer ejercicio", "Comprar pan"]

# 2. Agregamos una nueva tarea
tareas.append("Pasear al perro")

# 3. Imprimimos la primera tarea (índice 0)
print(f"Lo primero que debo hacer es: {tareas[0]}\n")

# 4. Imprimimos toda la lista usando un ciclo For
print("Mis tareas para hoy son:")
for tarea in tareas:
    print(f"- {tarea}")

Diccionarios

A diferencia de las listas, los diccionarios no usan números para encontrar sus elementos. Usan pares de clave-valor. Es igual que un diccionario real: buscas una palabra (la clave) para encontrar su significado (el valor). Se escriben entre llaves {}.

# Crear un diccionario de un usuario
usuario = {
    "nombre": "Martin",
    "edad": 22,
    "rol": "Estudiante"
}

# Acceder a un valor usando su clave
print(f"El usuario se llama {usuario['nombre']} y tiene {usuario['edad']} años.")

# Agregar una nueva clave-valor al diccionario
usuario["ciudad"] = "Santiago"
print("Diccionario completo:", usuario)
Práctica 2: Contacto de Emergencia

Crea un script donde definas un diccionario llamado contacto. Este diccionario debe tener:

  • La clave "nombre" con el nombre de tu contacto.
  • La clave "telefono" con su número.
  • La clave "parentesco" (por ejemplo: "Hermano", "Madre", "Amigo").

Luego, haz lo siguiente:

  1. Imprime un mensaje usando las claves para que diga algo como: "Llamar a [nombre] al [telefono]".
  2. Agrega una nueva clave llamada "email" al diccionario y asígnale un correo.
solucion_diccionarios.py 
# 1. Creamos el diccionario
contacto = {
    "nombre": "Lucas",
    "telefono": "+56912345678",
    "parentesco": "Amigo"
}

# 2. Imprimimos el mensaje accediendo a las claves
print(f"En caso de emergencia, llamar a {contacto['nombre']} al {contacto['telefono']}")

# 3. Agregamos el email
contacto["email"] = "lucas@ejemplo.com"
print("\nContacto actualizado:")
print(contacto)

Matrices (Listas de Listas)

En Python, no estás limitado a guardar datos simples dentro de una lista. ¡Puedes guardar listas dentro de otras listas! Esto crea lo que conocemos como una Matriz, que es como una tabla o un tablero con filas y columnas.

¿Cómo ubicamos un elemento? Primero le decimos a Python en qué fila está, y luego en qué columna. Por ejemplo: tablero[fila][columna]. ¡Recuerda que ambos empiezan a contar desde 0! 
# Creamos una matriz 3x3 (como un teclado de celular)
teclado = [
    [1, 2, 3],  # Fila 0
    [4, 5, 6],  # Fila 1
    [7, 8, 9]   # Fila 2
]

# Si queremos acceder al número 6:
# Está en la fila 1 (la segunda), y en la columna 2 (la tercera)
numero = teclado[1][2]
print(f"El número seleccionado es: {numero}")

# También podemos imprimir una fila completa
print(f"La primera fila es: {teclado[0]}")
Práctica 3: El Centro del Tablero

Imagina que estás haciendo un juego de "Gato". Tienes el siguiente tablero vacío representado por guiones "-":

tablero = [
    ["-", "-", "-"],
    ["-", "-", "-"],
    ["-", "-", "-"]
]

Tu misión es escribir un script que haga lo siguiente:

  1. Define el tablero en tu código.
  2. Modifica la lista para colocar una "X" exactamente en el centro del tablero.
  3. Coloca una "O" en la esquina inferior derecha.
  4. Usa un ciclo for para imprimir el tablero fila por fila y ver cómo quedó tu jugada.
solucion_matrices.py 
# 1. Definimos el tablero inicial
tablero = [
    ["-", "-", "-"],
    ["-", "-", "-"],
    ["-", "-", "-"]
]

# 2. Colocamos la "X" en el centro (Fila 1, Columna 1)
tablero[1][1] = "X"

# 3. Colocamos la "O" en la esquina inferior derecha (Fila 2, Columna 2)
tablero[2][2] = "O"

# 4. Imprimimos el tablero ordenado
print("Estado del juego:")
for fila in tablero:
    print(fila)
