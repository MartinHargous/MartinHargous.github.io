Condicionales: If, Elif, Else

Enséñale a tu programa a tomar decisiones basadas en diferentes situaciones.

¿Qué son los condicionales?

Hasta ahora, nuestros programas se ejecutaban de arriba hacia abajo, línea por línea. Pero, ¿qué pasa si queremos que el código haga una cosa si el usuario es mayor de edad, y otra distinta si es menor? Para crear estas "bifurcaciones" o caminos, usamos la instrucción if (si condicional).

http://googleusercontent.com/image_content/183
Para preguntar, necesitamos comparar:

Antes de usar el if, recuerda cómo comparar valores. El resultado siempre será un booleano (True o False).

  • == (Igual a) ¡Ojo! Un solo = es para asignar variables, dos == son para comparar.
  • != (Distinto de)
  • > / < (Mayor que / Menor que)
  • >= / <= (Mayor o igual / Menor o igual)

Camino A o Camino B (if y else)

La estructura básica es preguntar si algo es verdadero (if). Si lo es, ejecutamos un bloque de código. Si no lo es (else), ejecutamos un bloque alternativo.

edad = int(input("¿Cuántos años tienes? "))

if edad >= 18:
    print("Eres mayor de edad.")
else:
    print("Eres menor de edad.")

print("Este mensaje se imprime siempre, sin importar la edad.")

Múltiples opciones con elif

A veces, "blanco o negro" no es suficiente y necesitamos evaluar varias condiciones seguidas. elif significa "sino, si...". Puedes usar tantos elif como necesites, pero siempre van entre el if (que es el primero) y el else (que es el último, por si todo lo anterior falla).

nota = float(input("Ingresa tu calificación (1 al 7): "))

if nota == 7:
    print("¡Excelente, puntuación perfecta!")
elif nota >= 4:
    print("Muy bien, estás aprobado.")
elif nota >= 3:
    print("Casi, pero tienes que recuperar.")
else:
    print("Estás reprobado. ¡A estudiar más!")

El Desafío: "Cine Python"

Escribe un programa que pregunte al usuario su edad y determine el precio de su entrada de cine basándose en estas reglas:

  • Si es menor de 12 años, la entrada cuesta $5.
  • Si tiene entre 12 y 17 años (menor de 18), la entrada cuesta $7.
  • Si tiene 18 años o más, la entrada cuesta $10.

Al final, imprime el precio que debe pagar.

solucion.py 
edad = int(input("¡Bienvenido al cine! ¿Cuántos años tienes? "))

if edad < 12:
    precio = 5
elif edad < 18:
    precio = 7
else:
    precio = 10

print(f"El precio de tu entrada es de ${precio}.")
