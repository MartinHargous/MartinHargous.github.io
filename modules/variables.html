Variables y Tipos de Datos

Aprende cómo almacenar información y qué formas puede tomar en Python.

¿Qué es una variable?

Imagina una variable como una caja etiquetada donde guardas un valor. En Python, no necesitas especificar qué tipo de caja es; el lenguaje se da cuenta solo (esto se llama tipado dinámico).

¿Cómo se define?

El nombre de la variable va a la izquierda, el signo = en medio, y el valor a la derecha.

Hablando con el usuario: print() e input()

Para que tus programas no sean aburridos, necesitas comunicarte con la persona que los usa. Para eso usamos dos funciones fundamentales:

  • print(): Sirve para "imprimir" o mostrar mensajes y variables en la pantalla.
  • input(): Sirve para pedirle al usuario que escriba algo con su teclado. Lo que escriba, lo puedes guardar directamente en una variable.
Importante: Todo lo que el usuario escribe usando input(), Python lo interpreta como texto (str).
¿Cómo mezclar texto y variables en un print?

Tienes dos formas principales de hacerlo:

1. Usando comas (,)

Python unirá automáticamente todo poniendo un espacio entre cada elemento.

nombre = "Ana"
edad = 25
print("Hola", nombre, "tienes", edad, "años.") 
# Resultado: Hola Ana tienes 25 años.
2. Usando f-strings (¡La más recomendada!)

Pones una f antes de las comillas y encierras las variables entre llaves {}. Es mucho más limpio de leer.

nombre = "Ana"
edad = 25
print(f"Hola {nombre}, tienes {edad} años.")
# Resultado: Hola Ana, tienes 25 años.
# Ejemplo completo juntando todo:
nombre_visitante = input("¿Cuál es tu nombre? ")
print(f"¡Bienvenido a la academia de Python, {nombre_visitante}!")

Tipos de Datos Principales

Tipo Nombre Ejemplo
int Numero Entero edad = 25
float Decimal precio = 19.99
str Texto (String) nombre = "Python"
bool Booleano
Representa un estado lógico: Verdadero (True) o Falso (False). 		es_valido = True

Ejemplo:

# Definimos diferentes tipos de datos
usuario_nombre = "Martin"    # Esto es un str (cadena de texto)
usuario_edad = 22            # Esto es un int (numero entero)
pago_realizado = True        # Esto es un bool

# Imprimimos usando f-strings (muy útil)
print(f"Hola {usuario_nombre}, tu edad es {usuario_edad}")

# Podemos cambiar el valor cuando queramos
usuario_edad = int(input("Ingresa tu nueva edad: ")) # usamos int() para definir la edad como un numero entero
print(f"Feliz cumpleaños, ahora tienes {usuario_edad} años.")

El Desafío: "Ficha de Producto"

Crea un script en Python que pregunte al usuario las siguientes variables para un producto de tecnología:

  • nombre_producto (usa input() para pedirlo)
  • precio_original (conviértelo a decimal usando float())
  • stock_disponible (conviértelo a entero usando int())
  • en_oferta (un booleano, defínelo como True o False directamente)

Pista:

Usa la función print() para mostrar toda la información del producto al final usando f-strings.
solucion.py 
# 1. Pedimos los datos al usuario (y convertimos los tipos de datos)
nombre_producto = input("¿Cuál es el nombre del producto? ")
precio_original = float(input("¿Cuál es el precio? "))
stock_disponible = int(input("¿Cuánto stock hay disponible? "))
en_oferta = True  # Esto lo definimos manualmente por ahora

# 2. Mostramos el resultado en pantalla
print("\n--- FICHA DEL PRODUCTO ---")
print(f"Producto: {nombre_producto}")
print(f"Precio: ${precio_original}")
print(f"Stock: {stock_disponible} unidades")
print(f"¿En oferta?: {en_oferta}")
